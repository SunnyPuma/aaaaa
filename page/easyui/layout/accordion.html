Accordion for jQuery

Accordion is a part of easyui framework for jQuery. It lets you define your accordion component on web page more easily.

创建分类

  通过标签创建分类，给div标签添加一个名为'easyui-accordion'的类ID。

  1. <div id="aa" class="easyui-accordion" style="width:300px;height:200px;">  
  2.     <div title="Title1" iconCls="icon-save" style="overflow:auto;padding:10px;">  
  3.         <h3 style="color:#0099FF;">Accordion for jQuery</h3>  
  4.         <p>Accordion is a part of easyui framework for jQuery.   
  5.         It lets you define your accordion component on web page more easily.</p>  
  6.     </div>  
  7.     <div title="Title2" iconCls="icon-reload" selected="true" style="padding:10px;">  
  8.         content2  
  9.     </div>  
  10.     <div title="Title3">  
  11.         content3  
  12.     </div>  
  13. </div>  

  我们可以更改或修改面板的一些功能以后再重新创建它。

  1. $('#aa').accordion({  
  2.     animate:false  
  3. });

容器属性

名称 类型 说明 默认
width number 可折叠容器的宽度。 auto
height number 可折叠容器的高度。 auto
fit boolean 设置为true来设置可折叠组件尺寸适合它的父容器。 false
border boolean 是否显示边界的定义。 true
animate boolean 定义是否显示动画效果展开或折叠面板。 true

面板设置

  可折叠组件面板选项是inhirited附属面板,下面是添加属性:

名称 类型 说明 默认
selected boolean 设置为true来展开面板。 false

名称 参数 说明
onSelect title,index 当一个小组被选中。
onAdd title,index 当一个新的面板添加。
onBeforeRemove title,index 在一个面板被删除前事件,返回false取消删除操作。
onRemove title,index 当一个面板被移除。

名称 参数 说明
options none 返回选择的组件。
panels none 得到所有的面板。
resize none 调整可折叠组件。
getSelected none 得到选中的面板。
getPanel which 得到指定的面板。“which”参数可以是标题或索引的面板。
getPanelIndex panel 得到指定的面板指数。这个方法是可用的,从1.3版本以后。

下面的例子显示了如何获得选定的面板指数。 

var p = $('#aa').accordion('getSelected');  
if (p){  	
   var index = $('#aa').accordion('getPanelIndex', p);  	
   alert(index);  
}
select which 选择指定的面板。“which”参数可以是标题或索引的面板。
add options 添加一个新的面板。默认情况下,添加面板将成为选择。选择添加一个新的面板,通过“selected”属性,将其设置为false。

代码示例: 

$('#aa').accordion('add', {  	title: 'New Title',  	content: 'New Content',  	selected: false  });
remove which 删除指定的面板。“which”参数可以是标题或索引的面板。
