重写默认方法 $.fn.panel.defaults.

面板是用来作为一个容器对于其他内容。这是最基本的组件构建其他组件(如layout, tabs, accordion, 等等) I还提供内置的可折叠,可闭,允许最大化和允许最小化行为和其他定制的行为。面板可以很容易地嵌入到web页面的任何位置。

创建面板

1. 通过标签创建面板

通过标签创建更简单。添加'easyui-panel'类ID到 div 标签。

  1. <div id="p" class="easyui-panel" title="My Panel"   
  2.         style="width:500px;height:150px;padding:10px;background:#fafafa;"  
  3.         data-options="iconCls:'icon-save',closable:true,  
  4.                 collapsible:true,minimizable:true,maximizable:true">  
  5.     <p>panel content.</p>  
  6.     <p>panel content.</p>  
  7. </div>  

2. 创建面板程序

让我们创建面板右上角的的工具栏。

  1. <div id="p" style="padding:10px;">  
  2.     <p>panel content.</p>  
  3.     <p>panel content.</p>  
  4. </div>  
  5.   
  6. $('#p').panel({  
  7.   width:500,  
  8.   height:150,  
  9.   title: 'My Panel',  
  10.   tools: [{  
  11.     iconCls:'icon-add',  
  12.     handler:function(){alert('new')}  
  13.   },{  
  14.     iconCls:'icon-save',  
  15.     handler:function(){alert('save')}  
  16.   }]  
  17. });   

移动面板

调用'move'方法移动面板到新的位置。

  1. $('#p').panel('move',{  
  2.   left:100,  
  3.   top:100  
  4. });   

读取内容

当加载成功的时候让我们通过ajax加载面板内容并显示一些消息。

  1. $('#p').panel({  
  2.     href:'content_url.php',  
  3.     onLoad:function(){  
  4.         alert('loaded successfully');  
  5.     }  
  6. });  

名称 类型 说明 Default
id string 面板的id属性。 null
title string 标题文本显示在面板标题。 null
iconCls string 一个CSS类来显示一个16 x16图标面板。 null
width number 设置面板宽度。 auto
height number 设置面板的高度。 auto
left number 设置面板左位置。 null
top number 设置面板顶部的位置。 null
cls string 添加一个CSS类的面板。 null
headerCls string 添加一个CSS类面板标题。 null
bodyCls string 添加一个CSS类面板的主体。 null
style object 添加一个自定义的规范风格的面板。

代码示例来改变面板边框宽度: 

<div class="easyui-panel" style="width:200px;height:100px" data-options="style:{borderWidth:2}"></div>
{}
fit boolean 当真正的设置面板尺寸适合它的父容器。他下面的例子显示了一个面板,可以自动调整大小以最大化填充父容器
<div style="width:200px;height:100px;padding:5px">  	
     <div class="easyui-panel" style="width:200px;height:100px" data-options="fit:true,border:false">Embedded Panel</div>  
</div>
false
border boolean 是否显示面板定义边界。 true
doSize boolean 如果设置为true,面板将创建时调整和做布局。 true
noheader boolean 如果设置为true,那么将不会创建面板header。 false
content string 面板的正文内容。 null
collapsible boolean 定义是否显示折叠按钮。 false
minimizable boolean 是否显示最小化按钮 false
maximizable boolean 是否显示最大化按钮 false
closable boolean 是否显示关闭按钮 false
tools array,selector 自定义工具,可能的值:
1) 一个数组,每个元素包含iconCls和handler属性。
2) 一个选择器,该选择器显示工具

面板工具可以被声明与存在< div >标记: 

<div class="easyui-panel" style="width:300px;height:200px" title="My Panel" data-options="iconCls:'icon-ok',tools:'#tt'"></div>  
<div id="tt">
    <a href="#" class="icon-add" onclick="javascript:alert('add')"></a>
    <a href="#" class="icon-edit" onclick="javascript:alert('edit')"></a>
</div>

面板工具也可以通过数组定义: 

<div class="easyui-panel" style="width:300px;height:200px" title="My Panel" data-options="iconCls:'icon-ok',tools:[
{
  iconCls:'icon-add',
  handler:function(){
         alert('add')
      }
},{
  iconCls:'icon-edit',
  handler:function(){ 
         alert('edit') 
      }
}]"></div>
[]
collapsed boolean 定义面板在初始化的时候是否收缩。 false
minimized boolean 定义面板在初始化的时候是否最小化。 false
maximized boolean 定义面板在初始化的时候是否最大化。 false
closed boolean 定义面板在初始化的时候是否关闭。 false
href string 一个URL加载远程数据,然后显示在面板。注意,内容不会被加载,直到面板是开放和扩大。这是一个创建延迟加载面板有用的:实例 
<div id="pp" class="easyui-panel" style="width:300px;height:200px" data-options="href='get_content.php',closed:true">  </div>  
<a href="#" onclick="javascript:$('#pp').panel('open')">Open</a>
null
cache boolean True为缓存展板内容,从href载入。 true
loadingMessage string 当加载远程数据显示一个消息在面板。 Loading…
extractor function 定义了如何提取内容从ajax响应,返回提取数据。 
extractor: function(data){  	
    var pattern = /<body[^>]*>((.|[\n\r])*)<\/body>/im;  	
    var matches = pattern.exec(data);  	
    if (matches){  		
         return matches[1];	// only extract body content  	
    } else {  		
         return data;  	
    }  
}

名称 参数 说明
onLoad none 当加载数据的时候执行事件。
onBeforeOpen none 面板打开时执行事件,返回false停止打开。
onOpen none 在面板打开后执行事件。
onBeforeClose none 面板关闭时执行事件,返回false停止关闭。下面的面板不能关闭。
<div class="easyui-panel" style="width:300px;height:200px;" title="My Panel" data-options="onBeforeClose:function(){return false}">The panel cannot be closed.</div>
onClose none 在面板关闭执行事件。
onBeforeDestroy none 面板销毁前执行事件,返回false停止销毁。
onDestroy none 面板销毁后执行事件。
onBeforeCollapse none 面板收缩前执行事件,返回false停止收缩。
onCollapse none 面板收缩后执行事件。
onBeforeExpand none 面板展开前执行事件,返回false停止展开。
onExpand none 面板展开后执行事件。
onResize width, height 改变面板大小执行事件。
width: 新的外宽度
height: 新的外高度
onMove left,top 移动面板执行事件
left: 新左边的位置
top: 新的最高位置
onMaximize none 最大化后执行事件
onRestore none 在窗口已经恢复到它原来的大小执行事件。
onMinimize none 最小化后执行事件

名称 参数 说明
options none 返回选择属性。
panel none 返回外面板对象。
header none 返回面板标题对象。
body none 返回对象的面板的主体。
setTitle title 设置标题文本的标题。
open forceOpen 当forceOpen参数设置为true,面板打开绕过onBeforeOpen回调。
close forceClose 当forceClose参数设置为true,面板关闭绕过onBeforeClose回调。
destroy forceDestroy 当forceDestroy参数设置为true,面板是摧毁绕过onBeforeDestroy回调。
refresh href 刷新面板加载远程数据。如果“href”参数是指定的,它将覆盖旧的“href”属性。

代码示例: 

// open a panel and then refresh its contents.  
$('#pp').panel('open').panel('refresh');  

// refresh contents with a new URL.  
$('#pp').panel('open').panel('refresh','new_content.php');
resize options 设置面板的大小和做布局。选择对象包含以下属性:
width: 新的面板宽度
height: 新的面板高度
left: 新的面板左位置
top: 新的面板顶部的位置

代码示例: 

$('#pp').panel('resize',{width: 600,height: 400});
move options 移动到一个新位置的面板。选择对象包含以下属性:
left: 新的面板左位置
top: 新的面板顶部的位置
maximize none 根据它的容器的面板形成充满容器。
minimize none 最小化面板。
restore none 恢复最大化面板回到原来的大小和位置。
collapse animate 收缩的面板的主体。
expand animate 展开的面板的主体。
