创建选项卡

1. 创建选项卡通过标记

  创建标签标记是更简单,我们不需要写任何JavaScript代码。记得添加“easyui-tabs”类< div / >标记。每个选项卡面板创建通过子< div / >标记,使用一样的面板。

  1. <div id="tt" class="easyui-tabs" style="width:500px;height:250px;">  
  2.     <div title="Tab1" style="padding:20px;display:none;">  
  3.         tab1  
  4.     </div>  
  5.     <div title="Tab2" data-options="closable:true" style="overflow:auto;padding:20px;display:none;">  
  6.         tab2  
  7.     </div>  
  8.     <div title="Tab3" data-options="iconCls:'icon-reload',closable:true" style="padding:20px;display:none;">  
  9.         tab3  
  10.     </div>  
  11. </div>  

2. 以编程的方式创建选项卡

  现在我们创建选项卡以编程的方式,我们同时抓住“onSelect”事件。

  1. $('#tt').tabs({  
  2.     border:false,  
  3.     onSelect:function(title){  
  4.         alert(title+' is selected');  
  5.     }  
  6. });  

添加新选项卡面板

  添加一个拥有小工具栏的新选项卡窗格, 这个小工具栏图标(8x8) 被放置在关闭按钮之前。

  1. // add a new tab panel  
  2. $('#tt').tabs('add',{  
  3.     title:'New Tab',  
  4.     content:'Tab Body',  
  5.     closable:true,  
  6.     tools:[{  
  7.         iconCls:'icon-mini-refresh',  
  8.         handler:function(){  
  9.             alert('refresh');  
  10.         }  
  11.     }]  
  12. });  

得到选中的选项卡

  1. // get the selected tab panel and its tab object  
  2. var pp = $('#tt').tabs('getSelected');  
  3. var tab = pp.panel('options').tab;    // the corresponding tab object

选项卡面板

这个选项卡面板属性定义在面板组件,下面是一些常见的属性。

名称 类型 说明 默认
id string 选项卡面板的id属性。 null
title string 这个选项卡面板标题文本。
content string 这个选项卡面板内容。
href string 一个URL加载远程内容填充选项卡面板。 null
cache boolean 当为true是缓存选项卡面板,而且href属性设置有效。 true
iconCls string 一个图标的CSS类显示在选项卡面板标题。 null
width number 选项卡面板的宽度。 auto
height number 选项卡面板的高度。 auto

一些添加属性。

名称 类型 说明 默认
closable boolean 当设置为true,该选项卡面板将显示一个按钮就可闭点击关闭选项卡面板。 false
selected boolean 当设置为true,选项卡选项卡面板将被选择。 false

名称 参数 说明
onLoad panel 当ajax选项卡面板完成加载远程数据时执行事件。
onSelect title,index 当用户选择一个选项卡面板触发事件。
onBeforeClose title,index 在选项卡面板关闭,返回false取消这个动作。下面的例子显示了如何显示确认对话框之前关闭选项卡面板。 
$('#tt').tabs({
      onBeforeClose:function(title){ 
                        return confirm('Are you sure you want to close ' + title);    
                     }  
});  // using the async confirm dialog  

$('#tt').tabs({    
      onBeforeClose: function(title,index){  	
              var target = this;  	
              $.messager.confirm('Confirm',
                                 'Are you sure you want to close '+title,function(r){
                                                                    if (r){  			
                                                                          var opts = $(target).tabs('options');  			
                                                                          var bc = opts.onBeforeClose;  			
                                                                          opts.onBeforeClose = function(){};  // allowed to close now  			
                                                                          $(target).tabs('close',index);  			
                                                                          opts.onBeforeClose = bc;  // restore the event function 
                                                                          } });  	
                                   return false;	// prevent from closing    
               }  
});
onClose title,index 当用户关闭选项卡面板触发事件。
onAdd title,index 当一个新的选项卡面板添加触发事件。
onUpdate title,index 当一个选项卡面板更新触发事件。
onContextMenu e, title,index 当一个选项卡面板是正确的点击触发事件。

名称 参数 说明
options none 返回标签选项。
tabs none 返回所有选项卡面板。
resize none 调整标签容器和做布局。
add options

添加一个新的选项卡面板,选择参数是一个配置对象,看到选项卡面板属性为更多的细节。

当添加一个新的选项卡面板,它将成为选择。　　

添加一个未经选择的选项卡面板,记得设置”selected“属性为false。 

// add a unselected tab panel  
$('#tt').tabs('add',
        {
          title: 'new tab',
          selected: false 
          //... 
});
close which 关闭选项卡面板,‘which’参数可以是标题或索引的选项卡面板关闭。
getTab which 得到指定的选项卡面板,‘which’参数可以是标题或索引的选项卡面板。
getTabIndex tab 得到指定的选项卡面板指数
getSelected none 得到选中的选项卡面板。下面的例子显示了如何得到指数选择的选项卡面板。 
var tab = $('#tt').tabs('getSelected');  
var index = $('#tt').tabs('getTabIndex',tab);  
alert(index);
select which 选择一个选项卡面板,‘which’参数可以是标题或索引的选项卡面板。
exists which 表明如果特别面板是存在,‘which’参数可以是标题或索引的选项卡面板。
update param 更新指定的选项卡面板,param参数包含两个属性:
tab: 更新的选项卡面板。
options: 面板选项。

代码示例:
// 更新选中的面板与新标题和内容 
var tab = $('#tt').tabs('getSelected');  

// 得到选择面板 
$('#tt').tabs('update', {  	
    tab: tab,  	
    options: {  		
        title: 'New Title',  		
        href: 'get_content.php'  // the new content URL  	
    }  
});  
// 称之为'refresh'选项卡面板的方法来更新其内容  
var tab = $('#tt').tabs('getSelected');  

// 得到选择面板 
tab.panel('refresh', 'get_content.php');
enableTab which 使指定的选项卡面板,‘which’参数可以是标题或索引的选项卡面板。这个方法是可用的,从1.3版本以后。

代码示例： 

$('#tt').tabs('enableTab', 1);	// 使第二个选项卡面板
$('#tt').tabs('enableTab', 'Tab2');	//启用选项卡面板,有“Tab2”头衔
disableTab which 禁用指定的选项卡面板,‘which’参数可以是标题或索引的选项卡面板。这个方法是可用的,从1.3版本以后。

代码实例 : 

$('#tt').tabs('disableTab', 1);	// 禁用第二个选项卡面板。
scrollBy deltaX 滚动选项卡头由指定的数量的像素, 负值滚动到正确的、正值滚动到左边. T他的方法是可用的,从版本1.3.2开始。

代码示例: 

// 头向左滚动选项卡
$('#tt').tabs('scroll', 10);
