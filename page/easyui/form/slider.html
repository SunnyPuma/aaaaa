应用实例

当使用作为一个表单字段,创建滑块从<input>标记.

  1. <input class="easyui-slider" value="12"  style="width:300px"  
  2.         data-options="showTip:true,rule:[0,'|',25,'|',50,'|',75,'|',100]" />  

创建滑块从< div >也是允许的,但“value”属性是无效的.

  1. <div class="easyui-slider" data-options="min:10,max:90,step:10" style="width:300px"></div>  

以编程的方式创建滑块.

  1. <div id="ss" style="height:200px"></div>  
  1. $('#ss').slider({  
  2.     mode: 'v',  
  3.     tipFormatter: function(value){  
  4.         return value + '%';  
  5.     }  
  6. });  
名称 类型 说明 默认值
width number 滑块的宽度。 auto
height number 滑块的高度。 auto
mode string 显示什么类型的滑块。可能的值:“h”(水平)、“v”（垂直)。 h
reversed boolean 设置为true的最小值和最大值将开关他们的位置。这个属性可以从版本1.3.2开始使用。 false
showTip boolean 定义是否显示值信息提示。 false
disabled boolean 定义是否禁用滑块。 false
value number 默认值。 0
min number 允许的最小值。 0
max number 最大允许的值。 100
step number 增加或减少值。 1
rule array 显示标签旁边滑块,“|”——显示就行。 []
tipFormatter function 一个函数格式滑块值。返回字符串值,将显示作为提示。
名称 参数 说明
options none 返回滑块选项。
destroy none 销毁滑块对象。
resize param 设置滑块大小。“param”参数包含以下属性:
width: 新的滑块宽度
height: 新的滑块高度
getValue none 得到滑块值。
setValue value 滑块设置值。
enable none 启用滑块组件。
disable none 禁用滑块组件。
名称 参数 说明
onChange newValue, oldValue 当字段值变化时触发事件。
onSlideStart value 当开始拖动滑块时触发事件。
onSlideEnd value 当结束拖动滑块时触发事件。
