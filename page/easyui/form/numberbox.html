用法

创建numberbox从标记.

  1. <input type="text" class="easyui-numberbox" value="100" data-options="min:0,precision:2"></input>  

创建numberbox使用javascript.

  1. <input type="text" id="nn"></input>  
  1. $('#nn').numberbox({  
  2.     min:0,  
  3.     precision:2  
  4. });  

从validatebox属性扩展,下面是添加属性numberbox。

名称 类型 说明 默认值
disabled boolean 是否禁用该领域的定义。 false
value number 默认值。
min number 允许的最小值。 null
max number 最大允许的值。 null
precision number 最大的精度显示十进制分隔符之后。 0
decimalSeparator string 十进制字符分隔开的整数和小数部分的数。 .
groupSeparator string 字符分隔整数组显示千或百万。
prefix string 前缀字符串。
suffix string 后缀字符串。
formatter function(value) 一个函数的格式numberbox值。返回字符串值,将显示在输入框。
parser function(s) 一个函数来解析一个字符串。numberbox返回值。

从validatebox方法扩展,下面是对numberbox添加或重写的方法。

名称 参数 说明
options none 返回numberbox选项。
destroy none 销毁numberbox对象。
disable none 禁用字段。
enable none 启用字段。
fix none 修复真实有效值。
setValue value numberbox设置值。

代码示例:

$('#nn').numberbox('setValue', 206.12);
getValue none 得到numberbox值。

代码示例:

var v = $('#nn').numberbox('getValue');  alert(v);
clear none 清晰的numberbox值。
reset none numberbox重置价值。这个方法是可用的,从版本1.3.2开始使用。
名称 参数 说明
onChange newValue,oldValue 当字段值是变化的时候触发事件。
