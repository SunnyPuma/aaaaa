应用实例

创建timespinner从标记.

  1. <input id="ss" class="easyui-timespinner"  style="width:80px;"  
  2.         required="required" data-options="min:'08:30',showSeconds:true" />  

创建timespinner使用javascript.

  1. <input id="ss" style="width:80px;">  
  1. $('#ss').timespinner({  
  2.     min: '08:30',  
  3.     required: true,  
  4.     showSeconds: true  
  5. });  

属性扩展从spinner,下面是添加属性timespinner

名称 类型 说明 默认值
separator string 小时和分钟和秒之间的分隔符。 :
showSeconds boolean 定义是否显示秒。 false
highlight number 字段来强调最初, 0 = hours, 1 = minutes, … 0

从spinner的方法扩展,下面是timespinner覆盖方法。

名称 参数 说明
options none 返回选择对象。
setValue value 设置timespinner值。

代码示例:

$('#ss').timespinner('setValue', '17:45');  // set timespinner value  
var v = $('#ss').timespinner('getValue');  // get timespinner value  alert(v);
getHours none 获取当前的小时值。
getMinutes none 获取当前的分钟值。
getSeconds none 获得当前秒值。
