用法

组合可创建从一个<select>或<input>元素使用javascript。注意,创建组合从标记无效。

  1. <input id="cc" value="001">  
  1. $('#cc').combo({  
  2.     required:true,  
  3.     multiple:true  
  4. });  
从validatebox属性扩展,下面是添加属性组合。
名称 类型 说明 默认值
width number 组件的宽度。 auto
height number 组件的高度。这个属性可以从版本1.3.2使用。 22
panelWidth number 下拉面板宽度。 null
panelHeight number 下拉面板高度。 200
multiple boolean 定义是否支持多个选择。 false
separator string 分隔符字符文本当多个选择。 ,
editable boolean 是否用户可以输入文本定义直接到输入框。 true
disabled boolean 是否禁用该领域的定义。 false
hasDownArrow boolean 定义是否显示向下箭头按钮。 true
value string 默认值。
delay number 延迟做搜索,从最后一个关键的输入事件。 200
keyHandler object 一个函数被调用当使用者按下键。默认keyHandler被定义为: 
keyHandler: {  	
  up: function(){},  	
  down: function(){},  	
  enter: function(){},  	
  query: function(q){}  
}
从validatebox方法扩展,下面是添加的方法对组合。
名称 参数 说明
options none 返回选择对象。
panel none 返回下拉面板对象。
textbox none 返回文本框对象。
destroy none 销毁组件。
resize width 调整组件的宽度。
showPanel none 显示下拉面板。
hidePanel none 隐藏下拉面板。
disable none 禁用组件。
enable none 启用组件。
validate none 验证输入值。
isValid none 返回验证结果。
clear none 清空的组件值。
reset none 重置组件值。这个方法是可用的,从版本1.3.2以后。
getText none 得到输入文本。
setText text 设置文本值。
getValues none 获取组件值数组。
setValues values 设置组件值数组。
getValue none 获取组件值。
setValue value 设置组件的值。
名称 参数 说明
onShowPanel none 当下拉面板显示时触发事件。
onHidePanel none 当下拉面板隐藏时触发事件。
onChange newValue, oldValue 当字段值变化的时候触发事件。
