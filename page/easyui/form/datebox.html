属性扩展从combo,下面是添加属性datebox。

名称 类型 说明 默认值
panelWidth number 下拉日历面板宽度。 180
panelHeight number 下拉日历面板高度。 auto
currentText string 当前天按钮显示文本。 Today
closeText string 关闭按钮显示文本。 Close
okText string ok按钮显示文本。 Ok
disabled boolean 当为true则禁用字段。 false
formatter function 一个函数来格式化日期,该函数把一个“date”参数并返回一个字符串值。下面的例子显示了如何覆盖默认格式化程序功能。 
$.fn.datebox.defaults.formatter = function(date){  	
    var y = date.getFullYear();  	
    var m = date.getMonth()+1;  	
    var d = date.getDate();  	
    return m+'/'+d+'/'+y;  
}
parser function 一个函数来解析一个日期字符串,函数把一个“date”字符串,并返回一个日期值。下面的例子显示了如何覆盖默认的解析器函数。 
$.fn.datebox.defaults.parser = function(s){  	
   var t = Date.parse(s);  	
   if (!isNaN(t)){  		
       return new Date(t);  	
   } else {  		
       return new Date();  	
   }  
}

从combo的方法扩展,下面是datebox覆盖方法。

名称 参数 说明
options none 返回选择对象。
calendar none 把日历对象。下面的例子展示了如何把日历对象,然后重新创建它。 
//把日历对象  
var c = $('#dd').datebox('calendar');  

//设置第一天周星期一 
c.calendar({firstDay: 1});
setValue value 设置datebox值。

代码示例:

$('#dd').datebox('setValue', '6/1/2012');	// 给datebox赋值  
var v = $('#dd').datebox('getValue');	// 获得datebox的值
名称 参数 说明
onSelect date 当用户选择一个日期触发事件。

代码示例:

$('#dd').datebox({  	
   onSelect: function(date){  		
       alert(date.getFullYear()+":"+(date.getMonth()+1)+":"+date.getDate());  	
   }  
});
