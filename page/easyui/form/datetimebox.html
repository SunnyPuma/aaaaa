应用实例

创建datetimebox从标记.

  1. <input class="easyui-datetimebox" name="birthday"   
  2.         data-options="required:true,showSeconds:false" value="3/4/2010 2:3" style="width:150px">  

创建datetimebox使用javascript.

  1. <input id="dt" type="text" name="birthday"></input>  
  1. $('#dt').datetimebox({  
  2.     value: '3/4/2010 2:3',  
  3.     required: true,  
  4.     showSeconds: false  
  5. });  

从datebox属性扩展,下面是datetimebox添加属性:

名称 类型 说明 默认值
showSeconds boolean 定义是否显示第二个信息。 true
timeSeparator string 时间小时/分钟之间的分隔符和第二。这个属性可以从版本1.3以后使用。 :

从datebox方法扩展,下面是datetimebox覆盖方法。

名称 参数 说明
options none 返回选择对象。
spinner none 返回timespinner对象。
setValue value 设置datetimebox值。

代码示例:

$('#dt').datetimebox('setValue', '6/1/2012 12:30:56');	// 给datetimebox赋值  
var v = $('#dt').datetimebox('getValue');		// 获取datetimebox中的值;
