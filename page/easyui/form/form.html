用法

创建一个简单的HTML表单。构建这个像平时一样和id、action和method。

  1. <form id="ff" method="post">  
  2.     <div>  
  3.         <label for="name">Name:</label>  
  4.         <input class="easyui-validatebox" type="text" name="name" data-options="required:true" />  
  5.     </div>  
  6.     <div>  
  7.         <label for="email">Email:</label>  
  8.         <input class="easyui-validatebox" type="text" name="email" data-options="validType:'email'" />  
  9.     </div>  
  10.     ...  
  11. </form>  

使变成为ajax提交表单

  1. $('#ff').form({  
  2.     url:...,  
  3.     onSubmit: function(){  
  4.         // 做一些检查  
  5.         // 返回false来防止提交;  
  6.     },  
  7.     success:function(data){  
  8.         alert(data)  
  9.     }  
  10. });  
  11. // submit the form  
  12. $('#ff').submit();  

做一个提交动作

  1. // 称之为“submit”形式的方法插件提交表单  
  2. $('#ff').form('submit', {  
  3.     url:...,  
  4.     onSubmit: function(){  
  5.         // 做一些检查  
  6.         // 返回false来防止提交;  
  7.     },  
  8.     success:function(data){  
  9.         alert(data)  
  10.     }  
  11. });  

提交额外的参数

  1. $('#ff').form('submit', {  
  2.     url:...,  
  3.     onSubmit: function(param){  
  4.         param.p1 = 'value1';  
  5.         param.p2 = 'value2';  
  6.     }  
  7. });  

处理提交响应

提交一个ajax表单很简单。用户可以获得响应数据当提交完成。注意,响应的数据是原始数据从服务器。一个解析行动来响应数据需要得到正确的数据。

例如,响应数据被认为是JSON,典型的响应数据可能看起来像这样:

  1. {  
  2.     "success": true,  
  3.     "message": "Message sent successfully."  
  4. }  

现在处理JSON字符串在“成功”回调函数。

  1. $('#ff').form('submit', {  
  2.     success: function(data){  
  3.         var data = eval('(' + data + ')');  // change the JSON string to javascript object  
  4.         if (data.success){  
  5.             alert(data.message)  
  6.         }  
  7.     }  
  8. });  
名称 类型 说明 默认值
url string 表单动作URL提交 null
名称 参数 说明
onSubmit param 之前提交事件,返回false来防止提交动作。
success data 当表单提交successfuly触发事件。
onBeforeLoad param 在数据加载之前触发事件。取消这个动作返回false。
onLoadSuccess data 当表单加载数据触发事件。
onLoadError none 当一些错误发生而加载表单数据触发事件。
名称 参数 说明
submit options 做提交操作,选择参数是一个对象,包含以下属性:
url: action URL
onSubmit: 回调函数之前提交
success: 回调函数successfuly后提交

下面的例子显示了如何提交一个有效的形式和避免重复提交表单。

$.messager.progress();	// 显示进度条  
$('#ff').form('submit', {  	
   url: ...,  	
   onSubmit: function(){  		
        var isValid = $(this).form('validate');  		
        if (!isValid){  			
            $.messager.progress('close');	//隐藏进度条虽然形式是无效的 		
        }  		
        return isValid;	// 返回false,将阻止表单提交 	
   },  	
   success: function(){  		
        $.messager.progress('close');	// 隐藏进度条同时提交成功	
   }  
});
load data 加载记录填表。数据参数可以是一个字符串或一个对象类型,当字符串作为一个远程URL,否则作为一个当地的记录。

代码示例:

$('#ff').form('load','get_data.php');	// load from URL  
$('#ff').form('load',{  	
    name:'name2',  	
    email:'mymail@gmail.com',  	
    subject:'subject2',  	
    message:'message2',  	
    language:5  
});
clear none 清除表单数据.
reset none 重置表单数据。这个方法是可用的,从版本1.3.2开始。
validate none 做表单字段验证吗, 返回true当所有字段是有效的. 他的方法是用validatebox插件.
