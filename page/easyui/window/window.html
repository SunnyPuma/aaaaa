应用实例

创建一个窗口

1. 创建窗口从标记.

  1. <div id="win" class="easyui-window" title="My Window" style="width:600px;height:400px"  
  2.         data-options="iconCls:'icon-save',modal:true">  
  3.     Window Content  
  4. </div>  

2. 使用javascript创建窗口.

  1. <div id="win"></div>  
  1. $('#win').window({  
  2.     width:600,  
  3.     height:400,  
  4.     modal:true  
  5. });  

3. 创建窗口,complext布局.

像往常一样你可以定义窗口布局。下面的例子显示了如何分裂窗区域分为两部分:北部和中心.

  1. <div id="win" class="easyui-window" title="My Window" style="width:600px;height:400px"  
  2.         data-options="iconCls:'icon-save',modal:true">  
  3.     <div class="easyui-layout" data-options="fit:true">  
  4.         <div data-options="region:'north',split:true" style="height:100px"></div>  
  5.         <div data-options="region:'center'">  
  6.             The Content.  
  7.         </div>  
  8.     </div>  
  9. </div>  

一些请求与窗口

打开和关闭窗口.

  1. $('#win').window('open');  // open a window  
  2. $('#win').window('close');  // close a window  

通过Ajax请求加载一个窗口.

  1. $('#win').window('refresh', 'get_content.php');  

属性扩展从面板,下面是覆盖和添加属性窗口

名称 类型 说明 默认值
title string 窗口标题文本。 New Window
collapsible boolean 定义是否显示折叠按钮。 true
minimizable boolean 定义是否显示最小化按钮。 true
maximizable boolean 定义是否显示最大化按钮。 true
closable boolean 定义是否显示关闭按钮。 true
closed boolean 初始是否关闭 false
zIndex number 窗口的z轴坐标 9000
draggable boolean 定义是否窗口可以拖。 true
resizable boolean 定义是否窗口可以调整大小。 true
shadow boolean 如果设置为true,当窗口显示阴影将显示。 true
inline boolean 定义了如何保持窗户,true呆在它的父组件上，false去掉上面的所有元素。 false
modal boolean 定义如果窗口是一个模式窗口。 true

从面板的方法扩展,下面是为窗口添加的方法

名称 参数 说明
window none 返回外窗口对象。
hcenter none 中心窗口只有水平。这个方法是可用的,因为版本1.3.1。
vcenter none 中心窗口只有垂直。这个方法是可用的,因为版本1.3.1。
center none 中心窗口在屏幕上。这个方法是可用的,因为版本1.3.1。
