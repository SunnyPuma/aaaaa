用法

创建一个propertygrid在标记。注意,列已经内置,不需要再次声明它们。

  1. <table id="pg"></table>  
  1. $('#pg').propertygrid({  
  2.     url:'propertygrid_data.json',  
  3.     showGroup:true  
  4. });  

添加一个新行,propertygrid。

  1. var row = {  
  2.   name:'AddName',  
  3.   value:'',  
  4.   group:'Marketing Settings',  
  5.   editor:'text'  
  6. };  
  7. $('#pg').propertygrid('appendRow',row);  

行数据

propertygrid datagrid的扩展。它的行数据格式是datagrid一样。作为一个property行,以下字段是必需的:
name:字段名.
value: 被编辑的字段值.
group: 按字段分组.
editor: 编辑在编辑属性值.

行数据的例子:

  1. {"total":4,"rows":[  
  2.     {"name":"Name","value":"Bill Smith","group":"ID Settings","editor":"text"},  
  3.     {"name":"Address","value":"","group":"ID Settings","editor":"text"},  
  4.     {"name":"SSN","value":"123-456-7890","group":"ID Settings","editor":"text"},  
  5.     {"name":"Email","value":"bill@gmail.com","group":"Marketing Settings","editor":{  
  6.         "type":"validatebox",  
  7.         "options":{  
  8.             "validType":"email"  
  9.         }  
  10.     }}  
  11. ]}  

属性扩展从datagrid,下面是添加属性propertygrid。

名称 类型 说明 默认值
showGroup boolean 是否显示属性组定义。 false
groupField string 定义分组字段名。 group
groupFormatter function(group,rows) 定义了如何格式化组值。这个函数接受以下参数:
group: 该分组字段值。
rows:行属于它的分组。

扩展的方法从datagrid,下面是propertygrid添加方法。

名称 参数 说明
expandGroup groupIndex 扩大指定组。如果“groupIndex的参数没有分配,展开分组所有。
collapseGroup groupIndex 指定的组崩溃。如果“groupIndex的参数没有分配,全部折叠组。
