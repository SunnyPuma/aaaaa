应用实例

创建那么treegrid速度将在HTML标记。在大多数情况下,treegrid遵循相同的结构和格式是datagrid

  1. <table id="tt" class="easyui-treegrid" style="width:600px;height:400px"  
  2.         data-options="url:'get_data.php',idField:'id',treeField:'name'">  
  3.     <thead>  
  4.         <tr>  
  5.             <th data-options="field:'name',width:180">Task Name</th>  
  6.             <th data-options="field:'persons',width:60,align:'right'">Persons</th>  
  7.             <th data-options="field:'begin',width:80">Begin Date</th>  
  8.             <th data-options="field:'end',width:80">End Date</th>  
  9.         </tr>  
  10.     </thead>  
  11. </table>  

treegrid使用javascript创建

  1. <table id="tt" style="width:600px;height:400px"></table>  
  1. $('#tt').treegrid({  
  2.     url:'get_data.jsp',  
  3.     idField:'id',  
  4.     treeField:'name',  
  5.     columns:[[  
  6.         {title:'Task Name',field:'name',width:180},  
  7.         {field:'persons',title:'Persons',width:60,align:'right'},  
  8.         {field:'begin',title:'Begin Date',width:80},  
  9.         {field:'end',title:'End Date',width:80}  
  10.     ]]  
  11. });  
名称 类型 说明 默认值
idField string 定义了关键字段来标识一个树节点。要求。 null
treeField string 定义了树节点字段。必须的。 null
parentField string 定义了关键字段来标识一个树节点的parentId字段。要求。 _parentId
isLeaf string 定义了关键字段来标识一个树节点是否叶子节点，取值范围：0/1。0不是叶子节点，1是叶子节点。要求。 isLeaf
animate boolean 定义是否显示动画效果当节点展开或折叠。 false
loader function(param,success,error) 定义了如何加载数据从远程服务器。可以放弃这次行动返回false。这个函数接受以下参数:
param: 参数对象传递给远程服务器。
success(data): 这个回调函数会调用这些当检索数据成功。
error(): 这个回调函数会调用这些当未能检索数据。 		json loader
loadFilter function(data,parentId) 返回过滤数据显示。
名称 参数 说明
options none 返回选择的treegrid属性。
resize options treegrid设置大小,选择包含两个属性:
width: 新宽度
height: 新高度
fixRowHeight id 修复指定的行高。
loadData data 加载treegrid数据。
reload id 重新加载那么treegrid速度将数据。如果“id”参数是通过,重新加载指定的树行,否则重新加载所有树行。

代码示例:

$('#tt').treegrid('reload', 2);	// reload the row which value is equals to 2  
$('#tt').treegrid('reload');	// reload the all rows
reloadFooter footer 重新加载页脚数据。
getData none 获得加载数据。
getFooterRows none 获得页脚数据
getRoot none 得到根节点,返回节点对象
getRoots none 得到根节点,返回节点数组。
getParent id 得到父节点。
getChildren id 得到子节点。
getSelected none 得到选中的节点,并返回它,如果没有节点选择返回null。
getSelections none 得到所有选定的节点。
getLevel id 获取指定节点级别。
find id 找到指定的节点,并返回该节点的数据。
select id 选择一个节点。
unselect id 取消选择一个节点。
selectAll none 选择所有节点。
unselectAll none 取消选择所有节点。
collapse id 收缩的一个节点。
expand id 展开一个节点。
collapseAll id 所有节点收缩。
expandAll id 展开所有的节点。
expandTo id 展开从根到指定的节点。
toggle id 切换展开/折叠状态的节点。
append param 添加节点到父节点。“param”参数包含以下属性:
parent: 父节点id,如果不指定,作为根节点添加。
data: 数组,节点数据。
代码示例: 
// 附加一些节点选定的行
var node = $('#tt').treegrid('getSelected');  
$('#tt').treegrid('append',{
  	parent: node.id,  // 节点有一个“id”定义的值通过“idField”属性
  	data: [{
  		id: '073',
  		name: 'name73'
  	}]
  });
insert param 插入一个新节点到指定的节点。“param”参数包含以下属性:
before: id值的节点插入之前。
after: id值的节点插入后。
data: 新节点的数据。

代码示例:

// 插入一个新节点之前选中的节点  
var node = $('#tt').treegrid('getSelected');
  if (node){
  	$('#tt').treegrid('insert', {
  		before: node.id,
  		data: {
  			id: 38,
  			name: 'name38'
  		}
  	});
  }
这个方法是可用的,因为版本1.3.1。
remove id 删除一个节点和它的子节点。
pop id 取到和返回一个节点数据包括其子节点后删除。这个方法是可用的,因为版本1.3.1。
refresh id 刷新指定的节点。
update param 更新指定的节点。这个参数包含以下属性:
id: 节点标识,表明节点更新。
row: 新行数据。

代码示例:

$('#tt').treegrid('update',{
  	id: 2,
  	row: {
  		name: 'new name',
  		iconCls: 'icon-save'
  	}
  });
这个方法是可用的,因为版本1.3.1。
beginEdit id 开始编辑一个节点。
endEdit id 编辑节点结束。
cancelEdit id 取消编辑节点。
getEditors id 得到指定的行编辑器。每个编辑器有以下属性:
actions: 的动作编辑器可以做。
target: 目标编辑器jQuery对象。
field: 字段名。
type: 编辑器类型。
getEditor param 得到指定的编辑器,参数包含两个属性:
id: 行节点id。
field: 字段名。
名称 参数 说明
onClickRow row 当用户点击一个节点。
onDblClickRow row 当用户双击一个节点。
onClickCell field,row 当用户点击一个单元格。
onDblClickCell field,row 当用户双击一个单元格。
onBeforeLoad row, param 一个请求加载数据前触发事件,返回false取消这个加载。
onLoadSuccess row, data 一个请求加载数据成功后触发事件,
onLoadError arguments 当数据加载失败时触发,参数的参数是一样的“error”功能的jquery ajax。
onBeforeExpand row 节点扩展之前,返回false取消这个展开行为。
onExpand row 当节点扩展。
onBeforeCollapse row 在节点收缩,返回false取消这个收缩。
onCollapse row 当节点收缩。
onContextMenu e, row 当节点被右键点击。
onBeforeEdit row 当用户开始编辑一个节点。
onAfterEdit row,changes 当用户完成编辑。
onCancelEdit row 当用户取消编辑节点。
