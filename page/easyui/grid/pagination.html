使用标签创建分页控件。

  1. <div id="pp" class="easyui-pagination" data-options="total:2000,pageSize:10" style="background:#efefef;border:1px solid #ccc;"></div>  

使用Javascript创建分页控件。

  1. <div id="pp" style="background:#efefef;border:1px solid #ccc;"></div>  
  1. $('#pp').pagination({  
  2.     total:2000,  
  3.     pageSize:10  
  4. });  

让我们使用面板和分页插件来创建一个ajax分页。当用户选择一个新页面的时候，面板将显示指定页面的内容。

  1. <div id="content" class="easyui-panel" style="height:200px"  
  2.         data-options="href:'show_content.php?page=1'">  
  3. </div>  
  4. <div class="easyui-pagination" style="border:1px solid #ccc;"  
  5.         data-options="  
  6.             total: 2000,  
  7.             pageSize: 10,  
  8.             onSelectPage: function(pageNumber, pageSize){  
  9.                 $('#content').panel('refresh', 'show_content.php?page='+pageNumber);  
  10.             }">  
  11. </div>  

名称 类型 说明 Default
total number 总记录数,应创建时设置分页。 1
pageSize number 页面传值多少。 10
pageNumber number 显示页码当创建分页。 1
pageList array 用户可以改变页面传值多少。pageList的属性定义了页面传值大小可以改变。

代码实例: 

$('#pp').pagination({pageList: [10,20,50,100]});
[10,20,30,50]
loading boolean 定义数据是否是加载的 false
buttons array

定义了自定义按钮,每个按钮包含两个属性: 　　

iconCls:CSS类,它将显示一个背景图像
handler: 一个处理函数当按钮被单击

这个自定义按钮可以创建从标记: 

<div class="easyui-pagination" style="border:1px solid #ccc" data-options="total: 114,
            buttons: [ {iconCls:'icon-add',handler:function(){alert('add')}},'-',
                       {iconCls:'icon-save',handler:function(){alert('save')}}]">
</div>

定制按钮也可以使用javascript创建: 

$('#pp').pagination({
         total: 114,
         buttons: [{
                     iconCls:'icon-add',
                     handler:function(){
                           alert('add')
                     }
                   },'-',
                   {
                     iconCls:'icon-save',
                     handler:function(){
                           alert('save')
                     }
                   }]
});
null
showPageList boolean 定义是否显示页面列表。 true
showRefresh boolean 定义是否显示刷新按钮。 true
beforePageText string 显示一个标签在输入组件。 Page
afterPageText string 显示一个标签后输入组件。 of {pages}
displayMsg string 显示一个页面信息。 显示 {from} 到 {to} 一共 {total}成员

名称 参数 说明
onSelectPage pageNumber, pageSize 当用户选择一个新的页面。回调函数包含两个参数:
pageNumber: 新页面数量
pageSize: 新页面元素个数

代码示例: 

$('#pp').pagination({
         onSelectPage:function(pageNumber, pageSize){
                        $(this).pagination('loading');
                        alert('pageNumber:'+pageNumber+',pageSize:'+pageSize);  		
                        $(this).pagination('loaded');  	
                      }  
});
onBeforeRefresh pageNumber, pageSize 点击刷新按钮前触发事件,返回false取消刷新操作。
onRefresh pageNumber, pageSize 刷新后触发事件。
onChangePageSize pageSize 改变页面显示元素个数触发事件。

名称 参数 说明
options none 返回选择对象。
loading none 使分页成为正在加载状态。
loaded none 使分页成为以加载状态。
refresh options 刷新并显示页面信息。这个方法是可用的,从1.3版本以后。

代码实例 : 

$('#pp').pagination('refresh');	// refresh the pager bar information  
$('#pp').pagination('refresh',{	// change options and refresh pager bar information  	
                     total: 114,  	
                     pageNumber: 6  
});
select page 选择一个新页面。页面索引从1开始。这个方法是可用的,从1.3版本以后。

代码实例: 

$('#pp').pagination('select');	// refresh the current page  
$('#pp').pagination('select', 2);	// select the second page
