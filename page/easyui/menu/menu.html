创建菜单

通过标签创建菜单，给 div 标签添加一个名为'easyui-menu'的类ID。每个菜单项都通过 div 标签创建。我们可以添加'iconCls'属性来给菜单项定义一个图标显示到菜单项的左侧。添加'menu-sep'类ID菜单项将会生成一个菜单分割线。

  1. <div id="mm" class="easyui-menu" style="width:120px;">  
  2.     <div>New</div>  
  3.     <div>  
  4.         <span>Open</span>  
  5.         <div style="width:150px;">  
  6.             <div><b>Word</b></div>  
  7.             <div>Excel</div>  
  8.             <div>PowerPoint</div>  
  9.         </div>  
  10.     </div>  
  11.     <div data-options="iconCls:'icon-save'">Save</div>  
  12.     <div class="menu-sep"></div>  
  13.     <div>Exit</div>  
  14. </div>  

使用Javascript创建菜单项并监听'onClick'事件。

  1. $('#mm').menu({  
  2.     onClick:function(item){  
  3.         //...  
  4.     }  
  5. });  

显示菜单

在菜单被创建的时候它是隐藏和不可见的。调用'show'方法显示菜单。

  1. $('#mm').menu('show', {  
  2.   left: 200,  
  3.   top: 100  
  4. });  

菜单项

菜单项代表一个单独的项目,是显示在一个菜单。它包含以下属性:

名称 类型 说明 默认
id string 菜单项的id属性。
text string 项目的文本。
iconCls string 一个CSS类来显示一个16 x16图标项在左边。
href string 设置页面位置同时点击菜单项。
disabled boolean 定义是否禁用菜单项。 false
onclick function 单击菜单项而触发的函数.

菜单属性

名称 类型 说明 默认
zIndex number 菜单的z轴样式位置。 110000
left number 菜单左位置。 0
top number 菜单顶部位置。 0
minWidth number 最小宽度的菜单。这个属性可以从版本1.3.2开始使用。 120

名称 参数 说明
onShow none 在菜单显示的时候触发。
onHide none 在菜单隐藏的时候触发。
onClick item 当单击该菜单项的触发事件。下面的例子显示了如何处理所有菜单项点击: 
<div class="easyui-menu" data-options="onClick:menuHandler" style="width:120px;">  	
   <div data-options="name:'new'">New</div>  	
   <div data-options="name:'save',iconCls:'icon-save'">Save</div>  	
   <div data-options="name:'print',iconCls:'icon-print'">Print</div>  	
   <div class="menu-sep"></div>  	
   <div data-options="name:'exit'">Exit</div>  
</div>  
<script type="text/javascript">  	
   function menuHandler(item){  		
               alert(item.name)  	
               }  
</script>

名称 参数 说明
options none 返回选择对象。
show pos 在指定位置显示一个菜单。
pos参数有两个属性:
left: 新左边的位置。
top: 新的最高位置。
hide none 隐藏菜单。
destroy none 摧毁一个菜单。
getItem itemEl 把菜单项属性,包括一个“target”属性指示项目DOM元素。下面的例子显示了如何指定项的id:
<div id="mm" class="easyui-menu" style="width:120px">  	
   <div>New</div>  	
   <div id="m-open">Open</div>  	
   <div>Save</div>  
</div>  
var itemEl = $('#m-open')[0];  // the menu item element  
var item = $('#mm').menu('getItem', itemEl);  
console.log(item);
setText param 设置指定的菜单项的文本。“param”参数包含两个属性:
target: DOM对象,该菜单项将打开。
text: 字符串,新的文本值。

代码示例: 

var item = $('#mm').menu('findItem', 'Save');  
$('#mm').menu('setText', {  	
   target: item.target,  	
   text: 'Saving'  
});
setIcon param 设置指定的菜单项的图标。“param”参数包含两个属性:
target: DOM对象,菜单项。
iconCls: 新的图标类。

代码示例: 

$('#mm').menu('setIcon', {  	
    target: $('#m-open')[0],  	
    iconCls: 'icon-closed'  
});
findItem text 找到指定的菜单项,返回对象getItem一样。

代码示例: 

// 找到“开放”的菜单和禁用它 
var item = $('#mm').menu('findItem', 'Open');  
$('#mm').menu('disableItem', item.target);
appendItem options 添加一个新的菜单项,选择“options“显示新项属性。默认情况下,添加项目将成为一个顶级菜单项。为了添加一个子菜单项,“parent”属性应该设置为指定父项元素,已经有子条目。

代码示例: 

// 附加一个顶级菜单项
$('#mm').menu('appendItem', {  	
    text: 'New Item',  	
    iconCls: 'icon-ok',  	
    onclick: function(){
          alert('New Item')
    }  
});  
//添加一个子菜单项  
var item = $('#mm').menu('findItem', 'Open');  
//找到“开放”的菜单 
$('#mm').menu('appendItem', {  	
    parent: item.target,  // the parent item element  	
    text: 'Open Excel',  	
    iconCls: 'icon-excel',  	
    onclick: function(){
        alert('Open Excel')
    }  
});
removeItem itemEl 删除指定的菜单项。
enableItem itemEl 允许使用菜单项。
disableItem itemEl 禁用菜单项。
