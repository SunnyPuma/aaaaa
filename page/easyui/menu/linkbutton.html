创建按钮

使用标签创建按钮更加简单。

  1. <a id="btn" href="#" class="easyui-linkbutton" data-options="iconCls:'icon-search'">easyui</a>  

也可以使用Javascript创建按钮。

  1. <a id="btn" href="#">easyui</a>  
  2. $('#btn').linkbutton({  
  3.     iconCls: 'icon-search'  
  4. });  

处理按钮的点击

点击按钮会将用户引导到其他页面。

  1. <a href="otherpage.php" class="easyui-linkbutton" data-options="iconCls:'icon-search'">easyui</a>  

下面的示例提示了一个警告信息。

  1. <a href="#" class="easyui-linkbutton" data-options="iconCls:'icon-search'"  
  2.         onclick="javascript:alert('easyui')">easyui</a>  

绑定Click事件。

  1. <a id="btn" href="#" class="easyui-linkbutton" data-options="iconCls:'icon-search'">easyui</a>  
  1. $(function(){  
  2.     $('#btn').bind('click', function(){  
  3.         alert('easyui');  
  4.     });  
  5. });  

名称 类型 说明 默认
id string 这种组件的id属性。 null
disabled boolean True禁用按钮 false
plain boolean True显示一个普通的效应。 false
text string 按钮的文本。 ''
iconCls string 一个CSS类来显示一个16 x16图标在左边。 null
iconAlign string 位置的按钮图标。可能的值是: 'left','right'. 这个属性可以从版本1.3.2以后使用。 left


名称 参数 说明
options none 返回选择属性。
disable none 禁用按钮

代码示例: 

$('#btn').linkbutton('disable');
enable none 启用按钮

代码示例: 

$('#btn').linkbutton('enable');
